投影片版

1. 為什麼要做這個

懶人包

學校的新生導覽太過籠統
且不夠貼近數學系及學生
想要設計一個 屬於數學系學生 資訊詳盡的網站

還記得剛考上彰師時，想必都很期待未來的大學生活吧
很多人都會開始在網路上搜尋起關於彰師的一切
想要快點準備好成為大學生

但是

想必大家的熱情都被學校 極簡 風格的網站澆熄了吧

即使點進入看了裡面的資訊

想找學貸的資訊 要自己去學務處找
想找課程的資訊 看了課程大綱一堆專有名詞還是不瞭
想找宿舍的資訊 住宿組網站只有圖片介紹而且不會說大一住哪

說真的
小大一那時候靠自己Google到的資訊比裡面實用多了

更有趣的來了

學校網站的設計是給所有大一新生
所有大一新生
所有大一新生

嗯 意思就是實際連結感很低
充其量學校新生網只是一個學校交代事情的公布欄

想要聽數學系的課程 要自己去系上網站搜
想要知道數學系要不要上寶山 很抱歉學校不會說
想要知道來宿舍該帶什麼 學校只有告訴你宿舍有什麼設備

如果
有一個網站可以屬於彰師數學的學生使用
可以在上面交換資訊
可以知道關於數學系更實際的心得
可以有學長姊經驗分享大學會有什麼有趣的事
如果有 那不知道會為多少新生解惑

所以

我們要來設計一個 專屬數學系學生 的網站啦

2. 怎麼樣跟學校不同呢

懶人包

  1. 帳號制
  2. 主題實作按優先順序
  3. 主題由系上同學按需求提出
  4. 主題內容會向全系學生募集

首先 會蒐集各式各樣的主題以及想法

所蒐集的主題最後會排順序來決定實作的 優先順序
再來會確定該主題會以什麼樣的形式呈現
會做表單讓全系的學生填寫喔

會在網站設計成 會員帳號制 的

帳號制
將會採取註冊後經管理員審核的方式

3. 為什麼要採取帳號制

  1. [ 排他性 ] 只讓 數學系學生 使用
  2. 為了後端伺服器可辨識身分

4. 目前網站架構

每個主題都是一個子系統
這樣就可以分開來做了

更多關於建置詳情請點我

徵求

已列入考量主題

當然，這些不會是全部一次完成
將會按各個的優先順序一一實作出來
所以歡迎有任何想法就告訴我們，我們有空就會更新的
